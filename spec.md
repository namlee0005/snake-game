# Snake Game Web Project Spec

## Goal
Build a classic Snake Game playable in a web browser.

## Requirements
- Responsive game board (HTML5 Canvas).
- Score tracking and High Score (Local Storage).
- Increasing difficulty (speed increases as snake grows).
- Mobile-friendly controls (touch/swipe) + Keyboard support.
- Modern, clean UI using CSS.

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript.
- No heavy frameworks (keep it lightweight).
